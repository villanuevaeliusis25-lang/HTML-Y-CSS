📘 Apuntes de Formularios HTML

🎯 ¿Qué son los formularios?

Los formularios HTML permiten recoger datos de los usuarios y enviarlos a un servidor.

🏗️ Estructura Básica

<form action="/procesar-datos" method="POST">
    <!-- Aquí van los campos -->
</form>

Atributos principales:

  • action: URL donde se envían los datos
  • method: GET (visible en URL) o POST (oculto)

📝 Tipos de Campos de Input

Texto básico:

<input type="text" name="nombre" placeholder="Tu nombre">

Email con validación:

<input type="email" name="email" required>

Contraseña:

<input type="password" name="clave">

Número:

<input type="number" name="edad" min="0" max="120">

Fecha:

<input type="date" name="fecha_nacimiento">

🔘 Checkboxes y Radio Buttons

Checkbox (múltiple selección):

<label>
    <input type="checkbox" name="lenguajes" value="html"> HTML
</label>
<label>
    <input type="checkbox" name="lenguajes" value="css"> CSS
</label>

Radio Button (selección única):

<label>
    <input type="radio" name="genero" value="hombre"> Hombre
</label>
<label>
    <input type="radio" name="genero" value="mujer"> Mujer
</label>

📋 Select y Textarea

Lista desplegable (Select):

<label for="pais">País:</label>
<select id="pais" name="pais">
    <option value="es">España</option>
    <option value="mx">México</option>
    <option value="ar">Argentina</option>
</select>

Área de texto larga (Textarea):

<label for="mensaje">Mensaje:</label>
<textarea id="mensaje" name="mensaje" rows="4" cols="50"></textarea>

🛠️ Botones

<button type="submit">Enviar</button>
<button type="reset">Limpiar</button>
<button type="button">Botón normal</button>

✅ Atributos Importantes

  • required: Campo obligatorio
  • placeholder: Texto de ejemplo
  • min/max: Valores mínimo y máximo (para números/fechas)
  • minlength/maxlength: Longitud mínima/máxima del texto
  • disabled: Desactiva el campo

🔥 Ejemplo Completo

<form action="/registro" method="POST">
    <label for="nombre">Nombre completo:</label>
    <input type="text" id="nombre" name="nombre" required>

    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>

    <label for="password">Contraseña:</label>
    <input type="password" id="password" name="password" minlength="6">

    <button type="submit">Registrarse</button>
    <button type="reset">Limpiar</button>
</form>

💡 Recordatorios Importantes

  • ✅ Siempre usa el atributo name en los inputs
  • ✅ Usa <label> para mejor accesibilidad
  • ✅ El id del input debe coincidir con el for del label
  • ✅ Prueba los formularios con required y validaciones
